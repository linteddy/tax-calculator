# tax-calculator
South Africa income tax calculator
***

## PREREQUISITES TO BUILD AND RUN :
* Install java jdk 11+
* Install Angular
* Install maven

***

## RUN INSTRUCTIONS
* ### RUN SPRING-BOOT PROJECT
    * `cd server`
    * `mvn spring-boot:run`
* ### RUN ANGULAR PROJECT
    * `cd ui`
    * `ng serve --o`
